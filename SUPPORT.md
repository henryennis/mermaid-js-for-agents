# Support

Use GitHub issues for questions, bugs, and feature requests.

Before opening an issue:

1. Check whether the problem is renderer compatibility, Mermaid syntax, or package installation.
2. Include the target harness: Tessl, Pi, Claude Code, Codex, or another Agent Skills consumer.
3. Include the Mermaid diagram block or a minimal reproduction when safe.
4. Include the output of `npm run check` for package development problems.
